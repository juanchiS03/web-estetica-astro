# 🚀 Estética Sandra Martínez Córdoba

Este es el repositorio de la página web oficial para **Estética Sandra Martínez Córdoba**, un centro especializado en manicura, pedicura y bienestar ubicado en Logroño, La Rioja.

La web ha sido diseñada con un enfoque **"Mobile First"**, priorizando la velocidad de carga, la elegancia visual y la facilidad de reserva para las clientas.

---

## 🛠️ Tecnologías utilizadas

* **[Astro](https://astro.build/)**: Framework web moderno para conseguir una carga ultrarrápida mediante arquitectura de islas.
* **[Tailwind CSS](https://tailwindcss.com/)**: Framework de CSS para un diseño responsivo, moderno y totalmente personalizado.
* **[Vercel](https://vercel.com/)**: Plataforma de despliegue continuo (CI/CD) con optimización de bordes.
* **[Fontsource](https://fontsource.org/)**: Gestión optimizada de las tipografías *Playfair Display* y *Montserrat*.
* **Astro Sitemap**: Integración para la generación automática del mapa del sitio y optimización SEO en buscadores.

---

## ✨ Características principales

* ✅ **Diseño Responsivo**: Adaptado perfectamente a móviles, tablets y ordenadores (Bulletproof Layout).
* ✅ **Optimización de Imágenes**: Uso de formatos de última generación (`.webp`) y carga diferida (lazy loading).
* ✅ **SEO Local**: Configuración avanzada con Google Search Console para mejorar el posicionamiento en Logroño.
* ✅ **Estética Premium**: Paleta de colores basada en `slate-900` y tonos lila (`#CDB4DB`) para una sensación de lujo.
* ✅ **Rendimiento Máximo**: Puntuaciones optimizadas en Google Lighthouse.

## 📦 Instalación y Desarrollo

Si quieres clonar este proyecto y ejecutarlo en tu máquina local:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/nombre-de-tu-repo.git](https://github.com/tu-usuario/nombre-de-tu-repo.git)
    ```

2.  **Instala las dependencias:**
    ```bash
    npm install
    ```

3.  **Inicia el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

4.  **Construye el proyecto para producción:**
    ```bash
    npm run build
    ```
